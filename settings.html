📁 Google Drive — Photo & Document Storage

Not configured

Photos and documents attached to work orders are uploaded directly to your Google Drive folder. Free, no credit card required.

One-time setup (~10 min):

1. Go to console.cloud.google.com → New Project → name it anything

2. APIs & Services → Library → search Google Drive API → Enable

3. APIs & Services → OAuth consent screen → External → fill App name + your email → Save

4. APIs & Services → Credentials → Create Credentials → OAuth Client ID → Web application → add your site URL to "Authorized JavaScript origins" → Create → copy the Client ID

5. In Google Drive, create a folder for work order files → open it → copy the ID from the URL: drive.google.com/drive/folders/[FOLDER ID HERE]

All photos and PDFs will be uploaded into this folder.
